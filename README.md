# Curso Completo de SQL com SQL Server
## Do Básico ao Avançado

Este repositório reúne meus estudos completos sobre **SQL Server**, organizados de forma progressiva, do **nível iniciante ao avançado**, com foco em bancos de dados relacionais, boas práticas e uso profissional da linguagem SQL.

O material foi estruturado para servir tanto como **conteúdo de aprendizado** quanto como **referência técnica** para consultas futuras.

---

## Objetivo

O objetivo deste curso é aprender e consolidar o uso da linguagem **SQL (Structured Query Language)** utilizando o **SQL Server**, passando por todos os níveis de complexidade.

Ao final do estudo, espera-se ter domínio sobre:

- Criação e gerenciamento de bancos de dados
- Modelagem de dados
- Escrita de consultas simples e avançadas
- Otimização de desempenho
- Programação no banco de dados
- Boas práticas utilizadas no mercado

---

## Tecnologias Utilizadas

- SQL Server
- SQL Server Management Studio (SSMS)
- Linguagem SQL
- T-SQL (Transact-SQL)
